# Relax ON – Inclusive UI/UX Design Case Study

## Overview

Relax ON is a mental wellness and meditation mobile application designed using inclusive design principles. The project focuses on improving accessibility, usability, and user experience for users with diverse needs.

## Project Objective

To create an intuitive and accessible mobile application that promotes mental well-being while ensuring ease of use for all users.
